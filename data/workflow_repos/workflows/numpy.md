=== Description ===
NumPy is the core scientific computing library for Python, providing the N-dimensional array (ndarray) along with broadcasting, vectorized operations, and a large set of numerical routines (e.g., linear algebra, FFT, random numbers). It also includes tooling to integrate and accelerate code with C/C++ and Fortran extensions.
