=== Description ===
A browser-based web interface for running Stable Diffusion image generation locally, implemented with the Gradio UI framework. It provides txt2img/img2img workflows plus a large set of built-in features (inpainting/outpainting, upscaling, model/LoRA handling, training utilities, and an API) and supports community extensions.

=== Usage ===
Use this repository when working on generative-ai, stable-diffusion, diffusion-models tasks.
